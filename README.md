# HackersGens Flutter Project

Este repositório é dedicado ao meu progresso e desenvolvimento de atividades no curso de Flutter da **HackersGens**.

O objetivo principal deste projeto é acompanhar minha evolução prática durante o curso e cumprir os requisitos necessários para obtenção da badge de conclusão.

---

## Sobre o Projeto

Este projeto contém experimentações e implementações realizadas ao longo das aulas, incluindo:

* Estruturação de aplicativos Flutter
* Prática com widgets e layouts
* Navegação e organização de código
* Conceitos fundamentais de desenvolvimento mobile com Dart/Flutter

---

## Objetivo

* Consolidar aprendizado prático em Flutter
* Manter versionamento do progresso
* Registrar evidências para conclusão do curso
* Obter a badge oficial da HackersGens

---

## Tecnologias Utilizadas

* Flutter
* Dart
* Android/Web (targets de teste)
* Google IDX como IDE por conta da portabilidade

---

## Observações

Este repositório tem caráter educacional e experimental.
Não representa necessariamente um produto final ou aplicação pronta para produção.
